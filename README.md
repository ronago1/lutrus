Lutrus
======

A fork of the popular gaming interface for Linux called Lutris. This was created originally in response to GenAI being used in the development of Lutris.

Why does this exist?
--------------------

There are currently two main goals I want to achieve with this project that make it different from Lutris, though I would be happy to hear any additional ideas:

### 1. No AI written code
I will expand on this later in the document.

### 2. Qt interface
A Qt interface makes the desktop application fit with other apps on a Qt desktop environment. The Lutris developers don't have any plans for this currently (as far as I can tell).
