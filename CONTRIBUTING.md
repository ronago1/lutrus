Contributing to Lutrus
======================

Finding Stuff to Work On & Reporting Problems
---------------------------------------------

If you are looking for issues to work on or would like to bring attention in a problem, see the
[issues page](https://github.com/ronago1/lutrus/issues). If you are creating an issue, please
provide details to make things easier for anyone looking through the page.

You can also help by confirming (reproducing) or adding on to [existing issues](https://github.com/ronago1/lutrus/issues?q=is%3Aissue%20state%3Aopen).
You should verify that you can reproduce an issue before trying to fix it.

Please note that you can't create new features without first getting approval from ronago1.
I would be happy to hear about new feature ideas, but I don't want out-of-the-blue pull requests
for something I didn't know about.
